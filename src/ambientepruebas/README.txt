AmbientePruebas

Descripcion
Este proyecto está programado para llevar a cabo el proceso de control
de calidad de la página https://www.nacion.com/ como parte del curso de 
Calidad del Software, se desarrollarán pruebas funcionales al sitio en
análisis.

Instalación
Se realizó el proceso de instalación de Selenium WebDriver, para lo que
se añadieron las librerias JAR requeridas

Uso
El proyecto será usado para la ejecución de pruebas funcionales automáticas
a la página https://www.nacion.com/ con el fin de determinar el adecuado
funcionamiento del sitio

Estado del proyecto
Se encuentra en su estado inicial, la primera prueba ha sido realizada;
esta es la carga adecuada de la página de inicio de https://www.nacion.com/
